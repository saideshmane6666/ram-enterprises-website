# Ram Enterprises Website - Quick Start Guide

## 🚀 How to View the Website

### Option 1: Direct File Opening (Simplest)
1. Navigate to the website folder
2. Double-click `index.html`
3. Website opens in your default browser
4. Done! ✅

### Option 2: Using VS Code Live Server
1. Open VS Code
2. Open the `ram-enterprises-website` folder
3. Right-click on `index.html`
4. Select "Open with Live Server"
5. Website opens automatically
6. Changes auto-refresh in browser ✅

### Option 3: Web Server Upload
1. Upload all files to your web hosting
2. Maintain the exact folder structure
3. Visit your domain in browser
4. Website is live! ✅

### Changing Colors
1. Open `css/style.css`
2. Find `:root` at top
3. Modify color variables:
   - `--primary-blue: #1a7ac8` → Your color
   - `--accent-green: #22c55e` → Your color
4. All pages update automatically

### Adding New Page
1. Copy existing HTML page

3. Add link to navigation in all files

## 🚨 Troubleshooting

### Website Won't Open
- Check file path (no special characters)
- Ensure all files are in correct folders
- Try different browser
- Clear browser cache (Ctrl+Shift+Del)

### Links Not Working
- Verify file names match exactly
- Check file extensions (.html)
- Ensure proper folder structure
- Test with different browser

### Forms Not Submitting
- Check internet connection
- Enable JavaScript (usually enabled by default)
- Try different browser
- Check form field validation

### Maps Not Showing
- Check internet connection
- Verify iframe code intact
- Try different browser
- Clear cache

### Mobile Menu Not Working
- Enable JavaScript
- Check browser compatibility
- Clear cache
- Try different browser
