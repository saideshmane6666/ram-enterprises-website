## 🚀 Quick Start - Host on GitHub Pages

### 1️⃣ Prerequisites
- GitHub account (free at https://github.com)
- Git installed (https://git-scm.com)

### 2️⃣ Create Repository
- Go to https://github.com/new
- Name it: `ram-enterprises-website`
- Make it **Public**
- Click Create

### 3️⃣ Upload Files
```bash
cd path/to/ram-enterprises-website
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/ram-enterprises-website.git
git branch -M main
git push -u origin main
```

### 4️⃣ Enable GitHub Pages
1. Go to repository **Settings**
2. Click **Pages** (left sidebar)
3. Select **Deploy from a branch**
4. Choose `main` branch and `/(root)` folder
5. Save

✅ **Done!** Your site is live at: `https://YOUR_USERNAME.github.io/ram-enterprises-website`

📖 **Detailed Guide:** See [GITHUB_SETUP.md](GITHUB_SETUP.md)

## 🎨 Customization

### Change Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-blue: #1a3a52;
    --accent-green: #d4af37;
    /* More colors... */
}
```

### Add/Edit Content
Modify HTML files directly:
- `index.html` - Homepage content
- `services.html` - Service descriptions
- `about.html` - Company information
- `contact.html` - Contact details

### Update Translations
Edit `js/theme-language.js`:
```javascript
translations.en = {
    'Your Key': 'Your Value',
    // Add more translations
}
```

### Change Logo
Replace `logo.svg` with your custom logo
